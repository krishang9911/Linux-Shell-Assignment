# Shell Scripts Assignment

## Project Overview
This repository contains three shell scripts created for automating common tasks such as backup, file download, and system monitoring. These scripts are designed to run in Linux/WSL environments.

## Instructions for Running the Scripts
1. Open a terminal.
2. Navigate to the folder containing the scripts:
       cd "/path/to/Shell_Scripts"
3. Make the script executable(do this once per script)
      chmod +x Script_Name.sh
4> Run the scrpit
     ./Script_Name.sh

## Brief Description of Each Script

Backup_Script.sh – Automates backup of files from a specified directory.

Automated_Download_Script.sh – Downloads files automatically from a given URL.

CPU_Memory_Monitoring_Script.sh – Monitors and displays CPU and memory usage
